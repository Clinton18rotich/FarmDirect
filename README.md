# 🌾 FarmDirect - Africa's Agricultural Super App

**Connect Direct • Farm to Table**

## Overview
FarmDirect connects farmers directly to buyers, eliminating middlemen who take 40-60% of earnings.

## Features
- 🌽 Crop Marketplace
- 🐄 Livestock Trading
- 🚚 Uber-Style Delivery
- 💰 M-Pesa Payments with Escrow
- 🌤️ AI Weather Intelligence
- 📒 Blockchain Records
- 🧠 AI Price Prediction
- 🤝 Community Mediation
- 🏦 Farm Credit & Savings
- 🎓 Farming Academy

## Tech Stack
- Kotlin + Jetpack Compose
- MVVM + Clean Architecture
- Hilt (DI) + Retrofit + Room
